# ToDo-Apps

## Overview
 This project is a **To-Do List Management Application** built using **Express.js**, **SQLite**, and **Bootstrap5**. 
 It provides a user-friendly interface for managing tasks with essential CRUD functionalities. The application also 
 features user authentication using **Passport.js**, responsive design, and AJAX for dynamic updates.

 - First we need to install the dependencies using 
  ## npm i

  -We can run this Project using command 
  ## npm run dev
  -or
  ## node index.js
  
 
## Features
-**User Authentication**: 
  - Sign Up
  - Sign In
  - Session management with Passport.js
- **Task Management**:
  - Create, read, update, and delete tasks.
  - Dynamic task updates using AJAX.
- **Responsive Design**:
- Ensures compatibility with various screen sizes using Bootstrap 5.
- **Database**:
-  SQLite for lightweight and efficient storage.
- **Additional Functionalities**:
  - Edit buttons styled for better user experience.
  - Error handling for invalid inputs or failed authentication.
  - Secure password storage using hashing.

 ## Technologies Used
- **Backend**: Node.js with Express.js
- **Database**: SQLite (using Sequelize ORM)
- **Frontend**: Bootstrap 5 for responsive design
- **Authentication**: Passport.js for session-based authentication
- **Dynamic Updates**: AJAX for real-time task interactions
- **Template Engine**: EJS for server-side rendering
